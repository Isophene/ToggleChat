ToggleList

A list of all different Toggle Types and what they do

Toggle Name What it toggles
UHC "[00✫]" Turns ALL UHC chat on/off
Team "[TEAM]" Turns ALL Team chat on/off
Join "player joined." Turns ALL join notifications on/off
Leave "player left." Turns ALL leave notifications on/off
Guild "Guild >" Turns ALL guild chat on/off
Party "Party >" Turns ALL party chat on/off
Shout "[SHOUT]" Turns ALL shouts on/off
Housing "[OWNER]", "[CO-OWNER]", "[RES]" Turns ALL housing chat on/off
Global "Player: HELLO" Turns ALL general chat on/off
Colored Team "[YELLOW]", "[WHITE]", "[GREEN]", "[BLUE]", "[RED]" Turns ALL colored team chats on/off
Messages "To Player", "From Player", Turns ALL private messages on/off
Party Invites "player has invited you to join their party" Turns ALL party invites on/off
Spectator Chat "[SPECTATOR]" Turns ALL spectator chats on/off
Friend Requests "Friend request from player" Turns ALL friend request notifications on/off
Separators "---------------" Turns ALL line separators on/off
